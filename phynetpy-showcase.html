PhyNetPy: Advanced Phylogenetic Network Analysis

A comprehensive Python library for phylogenetic network methods

PhyNetPy provides powerful tools for network construction, manipulation, simulation, and inference in phylogenetic analysis. Built with performance and usability in mind, it brings advanced network algorithms to the Python ecosystem.

Core Methods & Capabilities

PhyNetPy offers a comprehensive suite of modules for phylogenetic network analysis

NET

Network

Core phylogenetic network data structures including Node, Edge, Graph, and Network classes. Provides the foundation for all network operations.

  • Node and Edge management
  • Network topology operations
  • Leaf and root identification
PAR

NetworkParser

Parse Nexus files and Newick strings to import existing phylogenetic networks. Supports standard file formats used in phylogenetic research.

  • Nexus file parsing
  • Newick string handling
  • Format validation
SIM

BirthDeath

Birth-death process network simulators including Yule and CBDP models. Generate synthetic networks for testing and research.

  • Yule model simulation
  • CBDP model support
  • Configurable parameters
UTIL

GraphUtils

Graph and network utility functions for topology analysis and manipulation. Essential tools for network comparison and analysis.

  • Topology analysis
  • Network comparison
  • Graph manipulation
MOV

NetworkMoves

Network topology modification operations including add/remove hybrid nodes and NNI (Nearest Neighbor Interchange) moves.

  • Hybrid node operations
  • NNI moves
  • Topology modification
GTR

GTR

Time-reversible substitution models including GTR, K80, F81, JC, HKY, K81, SYM, and TN93 for sequence evolution modeling.

  • Multiple substitution models
  • Model selection
  • Parameter estimation
MSA

MSA

Multiple Sequence Alignment parsing, storage, and analysis. Handle sequence data efficiently with optimized data structures.

  • Alignment parsing
  • Data compression
  • Sequence analysis
GEN

GeneTrees

Gene tree container and analysis utilities including consensus tree construction. Manage and analyze multiple gene trees.

  • Tree container management
  • Consensus construction
  • Tree analysis utilities
